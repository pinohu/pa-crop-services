# PA CROP Services — UX/UI/Copy/Conversion Audit
**Date:** March 23, 2026  
**Audited pages:** index.html, about.html, compliance-check.html, site.css, 404.html  

---

## CRITICAL — Conversion Killers (Fixed)

### 1. Fake Testimonials & Fabricated Social Proof
- **Issue:** "Michael R." / "Sarah K." / "David L." with 5-star reviews. Business is pre-launch — zero real clients. "99% client retention rate" is fabricated.
- **Risk:** FTC violations. Instant trust destruction if anyone checks. Competitors will screenshot this.
- **Fix:** Replaced testimonial cards with a credibility/trust section showing real, verifiable facts: CROP license, PA Notary, physical address, statute reference. No fake names, no fake stats.

### 2. Missing Favicon
- **Issue:** No `<link rel="icon">` on any page. Browser tab shows generic globe icon.
- **Risk:** Looks unfinished. Hurts perceived professionalism and repeat-visit recognition.
- **Fix:** Added inline SVG favicon (data URI) matching the nav-mark icon across all pages.

### 3. Missing og-image.jpg
- **Issue:** `og:image` references `/og-image.jpg` but file doesn't exist in public/.
- **Risk:** Social shares on LinkedIn, Facebook show blank or broken image.
- **Fix:** Flagged for asset creation (requires actual image file — generate via Canva/screenshot).

### 4. SSL Certificate Mismatch
- **Issue:** `pacropservices.com` (bare domain) throws SSL cert error. Cert appears valid only for Vercel's preview domain.
- **Risk:** Direct-typed URLs fail. Google may deindex. Trust badges meaningless without working HTTPS.
- **Fix:** Infrastructure fix needed in Vercel DNS settings. Not a code fix.

## HIGH — UX & Conversion (Fixed)

### 5. Hero Has Competing CTAs
- **Issue:** "See where you stand" + "Try the portal demo" split attention. Portal demo is a weaker CTA.
- **Fix:** Made "See where you stand" the dominant CTA. Changed secondary to "View pricing" (lower friction than demo). Added phone number as trust anchor below CTAs.

### 6. No "How It Works" Section
- **Issue:** Visitor lands, sees features and pricing but never understands the 3-step process.
- **Fix:** Added clean 3-step "How it works" section between features and pricing: (1) Check your entity, (2) Pick a plan, (3) We handle the rest.

### 7. Repeated Copy
- **Issue:** "No business owner should lose what they've built over a $7 filing they forgot about" appears in urgency section AND story blockquote — verbatim duplicate.
- **Fix:** Rewrote story blockquote to a different angle: "We built PA CROP Services because compliance shouldn't require a legal team."

### 8. Value Anchor Math Is Misleading  
- **Issue:** "What you'd pay separately" totals $1,347+/yr but includes "Basic business website — $500–2,000" which is a one-time cost, not annual. Mixing one-time and recurring costs inflates the number.
- **Fix:** Added "(one-time)" label next to website line. Adjusted total to show separate one-time vs annual math.

### 9. Emoji Icons in Feature Cards
- **Issue:** 🛡️⚡🔔📊📝 render differently on Windows/Android. Look unprofessional vs the rest of the design.
- **Fix:** Replaced all emoji with consistent inline SVG icons matching the design system colors.

### 10. No Phone Number in Nav
- **Issue:** For a local service business, phone builds instant trust. Phone is buried in footer only.
- **Fix:** Added phone number next to nav CTA on desktop.

## MEDIUM — UI Polish (Fixed)

### 11. About Page Uses Different Nav
- **Issue:** About page uses `site-nav` class (64px height) while homepage uses `nav` class (72px height). Different styling, different behavior. Jarring on navigation.
- **Fix:** Unified about page to use the shared site-nav from site.css consistently.

### 12. About Page CSS Variable Circular References
- **Issue:** About page `<style>` declares `--primary:var(--slate)` and `--accent:var(--gold-muted)` in its own `:root` — but these reference variables from site.css which may not resolve in the same cascade order.
- **Fix:** Replaced with direct color values from design system.

### 13. Compliance-Check Canonical URL Had .html Extension
- **Issue:** `<link rel="canonical">` pointed to `/compliance-check.html` but `vercel.json` has `cleanUrls: true`, so the canonical should be `/compliance-check`.
- **Fix:** ✅ Already fixed.

### 14. Welcome Page Uses Different Font Stack
- **Issue:** welcome.html loads Plus Jakarta Sans + Fraunces instead of Outfit + Instrument Serif. Brand inconsistency post-purchase.
- **Fix:** Flagged for separate update (lower priority since it's behind auth).

## RECOMMENDATIONS (Not Code Fixes)

### A. Generate Real og-image.jpg
Create a 1200×630px image with the PA CROP Services logo, tagline, and gold/slate colors. Upload to public/og-image.jpg.

### B. Fix Vercel SSL for Bare Domain
In Vercel dashboard → Settings → Domains: ensure both `pacropservices.com` and `www.pacropservices.com` are configured with proper SSL certs.

### C. Replace Fake Reviews When Real Ones Exist
The trust section I added is the right approach for pre-launch. As real clients sign up, replace with verified testimonials using real names and entity types.

### D. Consider 3-Tier Pricing
4 tiers creates decision fatigue for a new brand. Consider collapsing to 3 (Compliance / Starter / Pro) and offering Empire as a "Contact us" tier.

### E. Lead Magnet for Newsletter
"Send me the guide" promises a guide — make sure one actually gets delivered. Create a simple "PA Annual Report Compliance Checklist" PDF.
